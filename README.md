# LOST SIGNAL — 사랑과 평화를 찾아서

캐릭터 뒤를 따라가는 3D 시점으로 진행되는 대화형 영어 학습 어드벤처입니다.

## 실행

사진 인식은 Gemini API를 사용합니다. 전용 서버를 실행합니다.

```bash
python3 server.py
```

그런 다음 `http://127.0.0.1:4173`으로 접속하고 책 사진 화면 맨 위에 Gemini API 키를 입력합니다. 키는 현재 브라우저 탭에서만 유지되고 책 데이터에는 저장되지 않습니다. 모델을 바꾸려면 `GEMINI_MODEL` 환경 변수를 설정할 수 있으며, 기본값은 `gemini-3.5-flash`입니다.

## 게임 흐름

1. 3D 세계를 걸으며 스토리 대사를 만납니다.
   문제를 푸는 동안 캐릭터는 멈추고, 정답을 모두 맞히면 다음 장소로 다시 걷습니다.
2. 동사와 전치사를 찾아 표시합니다. 동사·전치사 단독 뜻 문제는 없습니다.
3. `주어+동사`의 뜻을 한 번에 맞힙니다.
4. 목적어·보어·수식어 등 나머지 부분의 뜻을 한 번에 맞힙니다.
5. 두 덩어리를 연결해 전체 해석과 문장 구조를 확인합니다.

키보드의 `A`·`D` 또는 방향키로 좌우를 살필 수 있고, `Space`로 점프할 수 있습니다.

## 숙제 사진으로 게임 만들기

1. 시작 화면에서 `책 사진 붙여넣기`를 누릅니다.
2. 표지 칸을 선택해 표지를 붙여넣으면 Gemini가 제목을 찾습니다.
3. 본문 칸을 선택해 페이지를 붙여넣으면 Gemini가 영어 문장, 한국어 해석, 동사, 전치사를 함께 분석합니다.
4. 완료할 때까지 본문 사진을 계속 추가한 뒤 문장을 저장합니다.

선택한 사진은 분석을 위해 Gemini API로 전송됩니다. Gemini의 인라인 이미지 요청은 사진 한 장당 20MB 미만으로 사용하세요.
